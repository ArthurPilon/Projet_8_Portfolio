Mon petit grimoire

J’ai développé un site web permettant aux utilisateurs de gérer une bibliothèque de livres en ligne. Ce projet a été une vraie découverte pour moi, car c’était ma première expérience avec le backend. J’ai notamment dû apprendre à créer des API REST, gérer les routes et connecter le tout à une base de données, ce qui a été un défi formateur.
